What is the project about ?

The goal of the project is to make a chat online that can gather student from Murdoch.

Since a lot of student from Murdoch are abroad, I wanted to create a chat around sharing languages and cultures within our campus. This chat will help people to learn new languages, share their own cultures and make new friends.

The main functionnalities of this chat will be simply : to create an account, register the courses you are in, select you native language, add friends based on your units and of course speak in a chat with a friend.

More functionnalities will be added such as creating a chat with multiple persons, adding documents in the chat, calling ...

See on github

Tech used

  • Node JS with Express for the server side
  • Vanilla JS and classic HTML using bootstrap for the client side
  • MongoDB for the database
  • Hosted by an AWS instance (EC2 Amazon)

My learning outcomes

  1. Learn how to use Web sockets for a real time web application
  2. Learn NoSQL queries with MongoDB
  3. Learn how to host a web application on a cloud instance
